This repository contains a collection of Java programs developed as part of weekly assignments. Each program focuses on core programming concepts such as loops, conditionals, type conversion, arrays, and logic building.

## 📌 Programs Overview

### 🔹 Week 3

**1. Marks Conversion**
- Converts data types:
  Float → Double → Integer → Character
- Demonstrates type casting in Java  

**2. ASCII Explorer**
- Takes a character input from user
- Shows ASCII value and modified character  

---

### 🔹 Week 4

**3. Fibonacci Series with Break**
- Prints Fibonacci series up to n terms  
- Stops when value exceeds 100  

**4. Unique Prime Factors**
- Prints unique prime factors of a number  
- Stops when factor exceeds n/2  

**5. Digit Transformer**
- Applies transformation rules on digits based on position  

---

### 🔹 Week 5

**6. Second Smallest Element**
- Finds second smallest number in array (pending implementation)
